# Map System

## Technology
- Leaflet.js
- OpenStreetMap

## Features
- Display stall markers
- Color-coded status:
  - Green: Active
  - Yellow: Expiring
  - Red: Violating
  - Gray: Inactive

## Interactions
- Click marker → popup
- Filter by category
- Filter by open now

## Location Validation
- Compare inspector GPS vs stall location
- Allow small radius
