# User Roles

## 1. Citizen
- View stalls on map
- Filter by category
- See images and schedules
- Report violations

## 2. Vendor (Stall Owner)
- Register and apply
- Upload stall images
- View assigned location
- View schedule
- Download QR code
- Manage subscription
- View violations

## 3. Officer (Inspector)
- Scan QR code
- Validate:
  - Location
  - Time
  - Activity
  - Subscription
- Record inspection
- Issue violations

## 4. Municipality Admin
- Manage stalls
- Approve/reject applications
- Manage locations
- Define rules
- Manage violations
- View reports
- Manage officers
