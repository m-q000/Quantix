# Problem & Solution

## Problem

Hebron suffers from:
- Unregulated street stalls
- Traffic and pedestrian congestion
- Conflicts with shop owners
- Lack of enforcement tools
- No structured schedule or zoning
- Lost municipal revenue
- No visibility for citizens

## Solution

A centralized system that:
- Maps approved stall locations
- Assigns vendors to locations
- Defines allowed activities per location
- Enforces time schedules
- Uses QR codes for instant inspection
- Allows citizens to browse stalls

## Key Transformation

From:
Uncontrolled street activity

To:
Digitally regulated micro-economy
