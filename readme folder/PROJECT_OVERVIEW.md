# Project Overview

## Name
Hebron Smart Stalls Regulation Platform

## Description
A municipality-driven web platform to regulate street vendors (stalls) in Hebron city.

The system transforms informal stalls into a structured, legal, and trackable system by:
- Assigning fixed locations
- Enforcing time schedules
- Controlling allowed activity types
- Enabling real-time inspection via QR codes
- Providing public visibility through an interactive map

## Objectives
- Reduce street chaos and congestion
- Prevent conflict with shops
- Increase municipality revenue
- Enable real-time inspection
- Provide transparency to citizens

## System Type
Multi-role web platform with 4 main portals:
- Public (Citizen)
- Vendor (Stall Owner)
- Officer (Inspector)
- Admin (Municipality)
