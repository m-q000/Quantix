# Business Rules

## Location Rules
- Each stall must be assigned to a predefined location
- GPS tolerance allowed (10–20 meters)
- No stall outside assigned location

## Activity Rules
- Each location has allowed categories
- Vendors cannot sell outside their category
- Conflict prevention with nearby shops

## Time Rules
- Each stall has fixed working days and hours
- No operation outside schedule

## Subscription Rules
- Active subscription required
- Expired = invalid stall

## QR Rules
- Each stall has unique QR
- Mandatory display
- Used for inspection

## Image Rules
- Max 3–5 images
- Must reflect actual products

## Violation Rules
- Violations logged by officers
- Repeated violations → suspension
